# 变更提案：会议预定与差旅出行智能化功能

## 变更名称
`add-smart-features`

## 状态
✅ 已完成

## 背景

### 问题描述
会议预定和差旅出行模块存在以下智能化不足：

**会议预定**：
- 已有用户偏好记录基础设施（userPreferenceRepo），但未在首页和预定页展示智能推荐
- 用户每次预定需要手动选择会议室，无法快速匹配常用会议室和空闲时段
- "一键秒定"功能设计文档中提及但未实现

**差旅出行**：
- 已有城市费用预估、模板复用功能
- 缺少出发前的行程提醒（天气、注意事项）
- 差旅结束后缺少报销流程支持，用户需手动整理费用

### 影响范围
- 首页（新增推荐卡片、提醒卡片）
- 会议预定页（新增悬浮推荐按钮）
- 差旅详情页（新增报销入口）
- 新增报销单填写页

## 动机

提升用户体验，减少重复操作：

1. **会议预定智能化**：根据历史偏好智能推荐会议室，一键完成预定
2. **差旅行程提醒**：出发前主动推送天气、注意事项，帮助用户做好准备
3. **差旅智能报销**：差旅结束后自动生成报销单预填，简化报销流程

## 目标

### 会议预定模块
- 首页展示 2 个推荐会议室卡片轮播
- 预定页悬浮按钮，点击弹出推荐列表
- 推荐因素：常用会议室 + 时段空闲 + 人数 + 设备 + 历史频率

### 差旅出行模块
- 出发前 1 天显示行程提醒卡片（天气、注意事项、建议携带）
- 差旅结束后显示报销提醒
- 支持一键生成报销单预填

## 非目标

- 不涉及报销审批流程（管理员端暂不实现）
- 不涉及真实天气 API 接入（使用城市配置表的天气提示）
- 不修改现有预定/差旅申请的业务逻辑
