# 极智协同 - 前端

![Deploy](https://github.com/zzp-997/howtoai/actions/workflows/deploy.yml/badge.svg)

企业 H5 移动端办公应用，支持员工和管理员两种角色，提供会议预定、差旅申请、请假打卡、文档查询、待办事项、公告通知、任务协作等功能。

## 技术栈

Vue 3 · Vite 6 · Pinia · TDesign Mobile · Vant 4 · Tailwind CSS · Axios

## 功能模块

### 一期 - 基础办公

| 模块 | 员工 | 管理员 |
|------|------|--------|
| 会议 | 预定会议室 | 管理会议室与预定 |
| 差旅 | 提交申请 | 审批申请 |
| 请假 | 提交申请 | 审批申请 |
| 考勤 | 打卡、补卡 | 查看考勤报表 |
| 文档 | 浏览文档 | 上传管理文档 |
| 待办 | 创建管理待办 | - |
| 公告 | 查看公告 | 发布管理公告 |
| 报销 | 提交报销 | 审批报销 |

### 二期 - 安全加固 + 协作增强

| 模块 | 说明 |
|------|------|
| 安全 | 双 Token、密码策略、登录限制、操作审计 |
| 审批 | 自定义审批链、批量审批、催办提醒 |
| 消息中心 | 审批通知、系统通知、任务通知统一管理 |
| 意见反馈 | 提交建议/Bug、进度查询 |
| 知识库 | 制度文档、技术指南、常见问题分类管理 |
| 数据统计 | 会议使用率、考勤分析、审批统计、综合大屏 |
| 任务协作 | 任务看板、子任务、评论、动态、关注 |

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式（前端运行在 localhost:3003）
npm run dev

# 生产构建
npm run build
```

## 部署

推送 `master` 分支自动触发 GitHub Actions 部署。
