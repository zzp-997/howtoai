## 一、设计概述

会议智能调度模块用于管理会议室预约和智能推荐功能，包含会议室信息、会议预约、智能推荐日志等表。

### 2.3 智能推荐日志表

```sql
CREATE TABLE schedule_recommendations (
    id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '推荐ID',
    user_id BIGINT NOT NULL COMMENT '用户ID',
    search_criteria JSON NOT NULL COMMENT '搜索条件',
    recommended_rooms JSON NOT NULL COMMENT '推荐的会议室列表',
    selected_room_id BIGINT COMMENT '用户最终选择的会议室ID',
    selected_start_time DATETIME COMMENT '用户选择的时间',
    is_accepted BOOLEAN DEFAULT FALSE COMMENT '是否采纳推荐',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_user (user_id),
    INDEX idx_created (created_at),
    INDEX idx_is_accepted (is_accepted),
    FOREIGN KEY (user_id) REFERENCES users(id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='智能推荐日志表';
```

### 2.4 会议室预约规则表

```sql
CREATE TABLE meeting_room_rules (
    id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '规则ID',
    room_id BIGINT NOT NULL COMMENT '会议室ID',
    rule_type ENUM('advance_booking', 'max_duration', 'min_duration', 'available_hours') NOT NULL COMMENT '规则类型',
    rule_value VARCHAR(100) NOT NULL COMMENT '规则值',
    is_active BOOLEAN DEFAULT TRUE COMMENT '是否启用',
    description VARCHAR(500) COMMENT '规则描述',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    INDEX idx_room (room_id),
    INDEX idx_type (rule_type),
    INDEX idx_active (is_active),
    FOREIGN KEY (room_id) REFERENCES meeting_rooms(id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='会议室预约规则表';
```

### 2.5 会议室可用时段表（缓存）

```sql
CREATE TABLE room_available_slots (
    id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT 'ID',
    room_id BIGINT NOT NULL COMMENT '会议室ID',
    slot_date DATE NOT NULL COMMENT '日期',
    start_time TIME NOT NULL COMMENT '开始时间',
    end_time TIME NOT NULL COMMENT '结束时间',
    is_available BOOLEAN DEFAULT TRUE COMMENT '是否可用',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    UNIQUE KEY uk_room_slot (room_id, slot_date, start_time, end_time),
    INDEX idx_room_date (room_id, slot_date),
    INDEX idx_available (is_available),
    FOREIGN KEY (room_id) REFERENCES meeting_rooms(id)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='会议室可用时段表';
```

## 四、视图设计

### 4.1 会议室预约详情视图

```sql
CREATE VIEW v_meeting_room_booking AS
SELECT
    m.id,
    m.title,
    m.room_id,
    mr.name AS room_name,
    mr.floor,
    mr.capacity,
    m.organizer_id,
    u.name AS organizer_name,
    m.attendees,
    m.start_time,
    m.end_time,
    m.status,
    m.is_online,
    m.meeting_link,
    TIMESTAMPDIFF(MINUTE, m.start_time, m.end_time) AS duration
FROM meetings m
LEFT JOIN meeting_rooms mr ON m.room_id = mr.id
LEFT JOIN users u ON m.organizer_id = u.id
WHERE m.deleted_at IS NULL;
```
