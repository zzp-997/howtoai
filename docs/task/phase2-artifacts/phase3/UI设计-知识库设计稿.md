## 一、页面概述

### 1.1 页面定位

知识库页面用于管理和展示企业知识文档，支持分类浏览、全文搜索、文档阅读等功能。

### 1.2 页面入口

工作台 → 知识库

## 三、组件设计

### 3.1 顶部搜索栏

| 组件 | 说明 |
|------|------|
| 搜索输入框 | 输入关键词搜索 |
| 搜索按钮 | 触发搜索 |
| 新建文档按钮 | 跳转新建文档页面 |
| 我的文档 | 筛选当前用户创建的文档 |

### 3.2 左侧分类栏

#### 3.2.1 分类树

- 树形结构展示分类
- 支持展开/折叠
- 支持拖拽排序
- 显示文档数量

#### 3.2.2 分类操作

| 操作 | 说明 |
|------|------|
| 新建分类 | 弹出新建分类弹窗 |
| 编辑分类 | 弹出编辑分类弹窗 |
| 删除分类 | 二次确认后删除 |

### 3.3 文档列表

#### 3.3.1 视图模式

- 卡片视图（默认）：显示文档卡片
- 列表视图：显示文档列表

#### 3.3.2 文档卡片

| 字段 | 说明 |
|------|------|
| 文档标题 | 标题文字，最多显示2行 |
| 文档摘要 | 内容摘要，最多显示3行 |
| 标签 | 显示最多3个标签 |
| 作者 | 发布人头像+姓名 |
| 阅读量 | 阅读次数 |
| 更新时间 | 最后修改时间 |

#### 3.3.3 文档列表项

| 字段 | 说明 |
|------|------|
| 标题 | 文档标题 |
| 分类 | 所属分类 |
| 作者 | 发布人 |
| 阅读量 | 数字 |
| 更新时间 | 日期 |
| 状态 | 草稿/已发布 |

#### 3.3.4 列表操作

- 筛选：全部/我的/已发布/草稿
- 排序：最新/最热/阅读量
- 批量操作：移动分类/删除/发布

### 3.4 文档详情页

#### 3.4.1 文档头部

| 字段 | 说明 |
|------|------|
| 标题 | 文档标题 |
| 作者信息 | 头像+姓名+发布时间 |
| 操作按钮 | 编辑/删除/分享/复制链接 |

#### 3.4.2 文档正文

- 富文本内容渲染
- 支持代码高亮
- 支持图片预览

#### 3.4.3 文档底部

| 区域 | 说明 |
|------|------|
| 标签区 | 显示文档标签 |
| 附件区 | 显示附件列表 |
| 相关信息 | 推荐相关文档 |

### 3.5 新建/编辑文档页面

| 字段 | 说明 |
|------|------|
| 标题输入框 | 最多200字符 |
| 分类选择 | 下拉选择 |
| 标签选择 | 多选标签 |
| 可见范围 | 单选：公开/部门/自定义 |
| 富文本编辑器 | 支持文字格式、图片、表格 |
| 发布/保存草稿 | 两个按钮 |

## 四、交互设计

### 4.1 搜索交互

- 搜索框获得焦点时，显示最近搜索历史
- 输入时实时联想
- 搜索结果高亮关键词
- 支持回车搜索

### 4.2 分类交互

- 点击分类展开/折叠子分类
- 点击分类加载对应文档列表
- 右键显示分类操作菜单
- 拖拽分类进行排序

### 4.3 文档操作

- 卡片悬停显示操作按钮
- 点击卡片跳转到详情页
- 右键显示文档操作菜单

### 4.4 编辑器交互

- 工具栏：文字格式、插入图片、插入表格
- 拖拽图片上传
- 拖拽文件上传
- 草稿自动保存提示
