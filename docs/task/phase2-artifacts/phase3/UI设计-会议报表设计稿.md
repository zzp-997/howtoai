## 一、页面概述

### 1.1 页面定位

会议报表页面用于展示会议室使用情况的统计数据，帮助管理者了解会议室利用率和会议安排情况。

### 1.2 页面入口

工作台 → 数据统计 → 会议报表

## 三、组件设计

### 3.1 顶部筛选器

| 组件 | 说明 |
|------|------|
| 时间类型切换 | 今日/本周/本月/本季度/本年，单选 |
| 自定义日期 | 点击展开日期范围选择器 |
| 会议室筛选 | 下拉多选，默认全选 |
| 导出按钮 | 导出Excel/PDF |

### 3.2 指标卡片

| 字段 | 说明 |
|------|------|
| 会议室总数 | 图标+数字+单位 |
| 平均使用率 | 数字+百分比+环比变化（↑绿/↓红） |
| 本期预约次数 | 数字+次+环比变化 |

### 3.3 趋势图（折线图）

- X轴：日期（按时间粒度）
- Y轴：使用率百分比
- 支持鼠标悬停显示详情
- 支持拖拽选择时间范围

### 3.4 使用对比图（柱状图）

- X轴：会议室名称
- Y轴：使用率百分比
- 按使用率降序排列
- 支持点击跳转到会议室详情

### 3.5 热门会议室列表

| 字段 | 说明 |
|------|------|
| 排名 | 1-5数字标识 |
| 会议室名称 | 文字 |
| 使用率 | 进度条+百分比 |
| 预约次数 | 数字+次 |

### 3.6 热力图

- 7列（周一到周日）
- 24行（0点到23点）
- 颜色深浅表示使用热度
- 白色=空闲，浅蓝=一般，深蓝=繁忙

## 四、交互设计

### 4.1 页面加载

- 骨架屏显示（3个卡片+2个图表占位）
- 数据加载顺序：指标卡片→趋势图→对比图→热力图

### 4.2 筛选交互

- 切换时间类型自动刷新数据
- 筛选变化时有loading态
- 筛选条件显示在筛选器旁边

### 4.3 图表交互

- 折线图：悬停显示具体数值
- 柱状图：点击可下钻到会议室详情
- 热力图：悬停显示具体时段使用情况

### 4.4 导出交互

- 点击导出弹出格式选择（Excel/PDF）
- 导出中显示loading
- 导出完成自动下载
