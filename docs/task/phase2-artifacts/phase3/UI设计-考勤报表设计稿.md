## 一、页面概述

### 1.1 页面定位

考勤报表页面用于展示员工的考勤统计数据，包括出勤率、请假情况、迟到早退等指标。

### 1.2 页面入口

工作台 → 数据统计 → 考勤报表

## 三、组件设计

### 3.1 顶部筛选器

| 组件 | 说明 |
|------|------|
| 时间类型切换 | 今日/本周/本月，单选 |
| 自定义日期 | 点击展开日期范围选择器 |
| 部门筛选 | 下拉单选，默认全公司 |
| 导出按钮 | 导出Excel |

### 3.2 指标卡片（6个）

| 指标 | 说明 | 图标 |
|------|------|------|
| 出勤率 | 百分比+环比 | 打卡图标 |
| 请假人次 | 数字+人次 | 请假图标 |
| 迟到人次 | 数字+人次+环比 | 时钟图标 |
| 早退人次 | 数字+人次+环比 | 时钟图标 |
| 缺勤人次 | 数字+人次+环比 | 警告图标 |
| 平均出勤时长 | 时:分 | 时长图标 |

### 3.3 趋势图（折线图）

- X轴：日期（按天）
- Y轴：出勤率百分比
- 支持多条折线对比（可选：各部门对比）
- 支持鼠标悬停显示详情

### 3.4 部门对比图（柱状图）

- X轴：部门名称
- Y轴：出勤率百分比
- 按出勤率降序排列
- 支持点击下钻到部门详情

### 3.5 请假类型分布（饼图/环形图）

| 类型 | 颜色 |
|------|------|
| 年假 | #3B82F6 |
| 病假 | #EF4444 |
| 事假 | #F59E0B |
| 其他 | #8B5CF6 |

### 3.6 考勤异常明细列表

| 字段 | 说明 |
|------|------|
| 员工姓名 | 文字 |
| 部门 | 文字 |
| 异常类型 | 标签（迟到/早退/缺勤） |
| 异常时间 | 具体时间 |
| 处理状态 | 标签（已处理/待处理） |
| 操作 | 处理按钮 |

分页：每页20条，支持翻页

## 四、交互设计

### 4.1 页面加载

- 骨架屏显示（6个卡片+3个图表+列表占位）
- 数据加载：指标→趋势图→对比图→饼图→列表

### 4.2 筛选交互

- 部门筛选变化时，图表和列表数据同时刷新
- 时间筛选变化时，重新计算统计数据

### 4.3 图表交互

- 折线图：悬停显示具体数值，支持图例点击切换显示
- 柱状图：点击可跳转部门考勤详情
- 饼图：悬停显示具体人数和占比

### 4.4 列表交互

- 点击处理按钮，弹出处理弹窗
- 支持排序（按时间/按部门）
- 支持搜索员工姓名
