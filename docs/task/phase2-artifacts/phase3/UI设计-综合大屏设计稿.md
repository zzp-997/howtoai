## 一、页面概述

### 1.1 页面定位

综合数据大屏（管理驾驶舱）是企业管理的核心数据展示页面，整合会议、考勤、审批等模块的关键指标，为管理者提供一站式数据概览。

### 1.2 页面入口

工作台 → 数据大屏

## 三、组件设计

### 3.1 顶部标题栏

| 元素 | 说明 |
|------|------|
| 页面标题 | "极智协同管理驾驶舱"，居中 |
| 当前日期 | 显示当前日期时间，实时更新 |
| 刷新按钮 | 手动刷新数据，刷新时显示loading |
| 全屏按钮 | 切换全屏显示模式 |
| 自动刷新 | 每5分钟自动刷新一次 |

### 3.2 核心KPI卡片

| 指标 | 说明 | 图标 | 颜色 |
|------|------|------|------|
| 今日会议 | 今日已预约+待预约会议数 | 会议图标 | #3B82F6 |
| 今日考勤 | 当前出勤率百分比 | 打卡图标 | #10B981 |
| 待审批数 | 当前积压的待审批总数 | 审批图标 | #F59E0B |
| 本月报销 | 本月报销总额（万元） | 报销图标 | #8B5CF6 |

卡片样式：
- 大数字突出显示
- 环比变化（上涨绿色箭头，下跌红色箭头）
- 卡片带有微光边框效果

### 3.3 会议概览模块

#### 3.3.1 会议室使用率TOP5

- 横向柱状图+进度条
- 显示会议室名称、使用率百分比、预约次数
- 点击可跳转到会议报表详情

#### 3.3.2 今日会议时间轴

- 时间轴形式展示今日会议安排
- 显示：时间、会议室、主题、预订人
- 当前时间有指示线
- 可点击跳转到会议详情

### 3.4 考勤概览模块

#### 3.4.1 出勤率趋势图

- 折线图显示近7天出勤率趋势
- 显示平均线
- 悬停显示具体数值

#### 3.4.2 考勤异常分布

- 环形图显示异常类型分布
- 分类：迟到、早退、缺勤、请假
- 悬停显示具体人数和百分比
- 点击跳转到考勤报表详情

### 3.5 审批概览模块

#### 3.5.1 待我审批列表

- 显示待审批事项
- 字段：类型、申请人、提交时间、紧急程度
- 支持快速审批（通过/驳回）

#### 3.5.2 我已审批列表

- 显示近期已审批事项
- 字段：类型、申请人、审批结果、审批时间

#### 3.5.3 超时审批列表

- 显示超时未处理的审批
- 红色高亮显示

#### 3.5.4 催办提醒

- 显示被催办的审批事项
- 字段：主题、催办人、催办时间

## 四、交互设计

### 4.1 页面加载

- 进入页面显示全屏loading动画
- 各模块依次加载，带入场动画
- 模块加载顺序：KPI→会议概览→考勤概览→审批概览

### 4.2 数据刷新

- 页面进入时自动加载最新数据
- 顶部刷新按钮手动刷新
- 刷新时各模块有局部loading效果
- 支持Ctrl+R快捷刷新

### 4.3 全屏模式

- 点击全屏按钮进入全屏
- ESC键退出全屏
- 全屏时隐藏系统UI

### 4.4 卡片点击

- KPI卡片点击可下钻到对应详情页面
- 列表项点击可跳转到详情页

### 4.5 快速操作

- 待审批列表支持快速通过/驳回
- 会议项点击可查看详情
- 支持快捷跳转功能

## 五、视觉规范

### 5.1 整体风格

- 深色主题（数据大屏专用）
- 背景色：#0F172A（深蓝黑）
- 卡片背景：rgba(30, 41, 59, 0.8) 半透明
- 边框：1px solid rgba(59, 130, 246, 0.3)

### 5.2 配色方案

| 元素 | 颜色 |
|------|------|
| 背景 | #0F172A |
| 卡片背景 | rgba(30, 41, 59, 0.8) |
| 主色 | #3B82F6 |
| 成功色 | #10B981 |
| 警告色 | #F59E0B |
| 错误色 | #EF4444 |
| 文字主色 | #F8FAFC |
| 文字次要 | #94A3B8 |

### 5.3 字体

| 元素 | 字号 | 字重 |
|------|------|------|
| 页面标题 | 28px | 700 |
| KPI数字 | 36px | 700 |
| KPI标签 | 14px | 400 |
| 模块标题 | 18px | 600 |
| 列表文字 | 14px | 400 |
| 辅助文字 | 12px | 400 |

### 5.4 动画效果

- 页面入场：模块依次淡入（stagger 100ms）
- 数字跳动：数字变化时有滚动动画
- 边框发光：卡片边框有微光流动效果

## 六、响应式设计

### 6.1 断点设计

| 断点 | 宽度 | 布局调整 |
|------|------|----------|
| 大屏 | >= 1920px | 16:9标准布局 |
| 标准 | 1440px - 1919px | 缩放适配 |
| 平板 | 1024px - 1439px | 改为2行布局 |
| 移动 | < 1024px | 单列堆叠布局 |

### 6.2 移动端适配

- 隐藏部分次要信息
- KPI卡片改为2x2布局
- 图表改为更紧凑的形式
- 列表改为卡片形式
