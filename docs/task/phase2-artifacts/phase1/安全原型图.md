## 一、原型概述

本文档定义「极智协同」企业OA办公应用安全功能的交互原型，涵盖三个核心安全流程的页面结构与交互设计。

### 原型范围

| 序号 | 功能模块 | 原型内容 |
|------|----------|----------|
| 1 | 登录失败限制 | 登录页错误提示、锁定提示、倒计时显示 |
| 2 | 密码强度校验 | 注册页密码强度指示器、修改密码页强度校验 |
| 3 | 密码过期提醒 | 登录成功后的过期提醒弹窗、强制修改密码页 |

### 2.3 关键元素说明

| 元素 | 类型 | 状态 | 交互行为 | 文案 |
|------|------|------|----------|------|
| 警告提示框 | Toast | 失败时显示 | 自动消失（5秒）或手动关闭 | "用户名或密码错误，还剩 X 次尝试机会" |
| 锁定提示框 | Alert | 锁定时显示 | 持续显示直至解锁 | "账户已锁定，请 X 分钟后重试" |
| 登录按钮 | Button | 正常/禁用 | 锁定状态禁用，其他状态可点击 | "登录" |
| 倒计时 | Text | 锁定时显示 | 每分钟更新 | "倒计时: XX:XX" |
| 密码输入框 | Input | 错误时高亮 | 错误时边框变红 | - |

### 2.4 状态说明

| 状态 | 触发条件 | UI表现 | 用户可操作 |
|------|----------|--------|-----------|
| 正常 | 初始状态 | 无警告提示 | 可输入、可登录 |
| 失败提示 | 登录失败1-4次 | 红色警告框，显示剩余次数 | 可输入、可登录 |
| 锁定状态 | 登录失败5次 | 红色锁定框，登录按钮禁用，显示倒计时 | 仅可查看、联系管理员 |
| 锁定期间尝试 | 锁定期内再次登录 | 提示剩余锁定时间，登录按钮可用 | 可输入、可登录（解锁后生效） |
| 解锁成功 | 锁定时间到期 | 恢复正常状态 | 可输入、可登录 |

### 3.3 关键元素说明

| 元素 | 类型 | 显示位置 | 交互行为 |
|------|------|----------|----------|
| 密码输入框 | Input | 顶部 | 输入时实时触发校验 |
| 强度指示器 | Segment | 输入框下方 | 动态显示当前强度 |
| 规则清单 | Checklist | 强度指示器下方 | 实时更新各规则状态 |
| 建议提示 | Text | 规则清单下方 | 根据未满足规则给出建议 |
| 提交按钮 | Button | 底部 | 强度达标后可点击 |
| 眼睛图标 | Icon | 输入框右侧 | 切换密码明文/密文显示 |

### 3.4 强度计算规则

| 规则项 | 权重 | 得分范围 |
|--------|------|----------|
| 长度 >= 8位 | 20分 | 0-20 |
| 包含大写字母 | 15分 | 0-15 |
| 包含小写字母 | 15分 | 0-15 |
| 包含数字 | 15分 | 0-15 |
| 包含特殊字符 | 20分 | 0-20 |
| 不包含用户名 | 10分 | 0-10 |
| 非常见弱密码 | 5分 | 0-5 |

**强度等级划分:**

| 等级 | 得分范围 | 颜色 | 是否允许提交 |
|------|----------|------|-------------|
| 弱 | 0-59 | 红色 | 否 |
| 中 | 60-79 | 黄色 | 是 |
| 强 | 80-100 | 绿色 | 是 |

### 3.5 状态说明

| 状态 | 触发条件 | UI表现 | 用户可操作 |
|------|----------|--------|-----------|
| 初始 | 未输入密码 | 强度指示器置灰，所有规则未勾选 | 可输入 |
| 输入中 | 正在输入 | 实时更新强度和规则状态 | 可输入、可切换显示 |
| 弱密码 | 得分<60 | 红色指示，提交按钮禁用 | 可继续输入 |
| 中等密码 | 得分60-79 | 黄色指示，提交按钮可用 | 可提交或继续优化 |
| 强密码 | 得分>=80 | 绿色指示，提交按钮可用 | 可提交 |
| 校验中 | 后端验证历史密码 | 显示loading，提交按钮禁用 | 等待结果 |

## 五、公共组件定义

### 5.1 警告提示组件 (Alert)

```
组件名称: Alert
用途: 显示登录失败提示、锁定提示等警告信息

属性:
- type: 'error' | 'warning' | 'info' | 'success'
- message: 提示文案
- icon: 是否显示图标 (boolean)
- closable: 是否可关闭 (boolean)
- duration: 自动消失时间(秒)，0为不消失

样式变体:
+----------------------------------+
| [!] 用户名或密码错误              |  type: error (红色)
+----------------------------------+

+----------------------------------+
| [锁] 账户已锁定，请15分钟后重试   |  type: error + icon (红色)
+----------------------------------+

+----------------------------------+
| [i] 密码将于5天后过期             |  type: warning (黄色)
+----------------------------------+
```

### 5.2 密码强度指示器组件 (PasswordStrengthIndicator)

```
组件名称: PasswordStrengthIndicator
用途: 显示密码强度等级

属性:
- strength: 'weak' | 'medium' | 'strong'
- score: 0-100 (可选，用于过渡动画)

样式变体:
弱密码 (0-59分):
+--------+--------+--------+
|  弱    |        |        |
+--------+--------+--------+
  红      灰      灰

中等密码 (60-79分):
+--------+--------+--------+
|  弱    |   中   |        |
+--------+--------+--------+
  灰      黄      灰

强密码 (80-100分):
+--------+--------+--------+
|  弱    |   中   |   强   |
+--------+--------+--------+
  灰      灰      绿
```

### 5.3 密码规则检查清单组件 (PasswordChecklist)

```
组件名称: PasswordChecklist
用途: 显示密码规则满足状态

属性:
- rules: Array<{ key: string, label: string, passed: boolean }>
- showAll: 是否显示全部规则 (boolean)

示例渲染:
+------------------------------------+
| [ok] 至少8位字符                    |
| [ok] 包含大写字母                   |
| [ok] 包含小写字母                   |
| [--] 包含数字                       |
| [--] 包含特殊字符                   |
| [ok] 不包含用户名                   |
| [ok] 非常见弱密码                   |
+------------------------------------+

图标说明:
- [ok]: 绿色勾选图标 (规则已满足)
- [--]: 灰色空心图标 (规则未满足)
- [err]: 红色叉号图标 (规则违反，如包含用户名)
```

### 5.4 倒计时组件 (CountdownTimer)

```
组件名称: CountdownTimer
用途: 显示锁定剩余时间

属性:
- targetTime: 目标时间 (Date)
- format: 'mm:ss' | '分钟' | '秒'
- onEnd: 倒计时结束时回调

样式变体:
+------------------------+
|    倒计时: 14:58       |
+------------------------+

+------------------------+
|    请 14 分钟后重试     |
+------------------------+

交互行为:
- 每秒更新显示
- 到达目标时间时触发 onEnd 回调
- 样式可能带有呼吸动画效果
```

## 八、边界场景处理

### 8.1 网络异常

| 场景 | UI处理 |
|------|--------|
| 登录请求超时 | 显示"网络异常，请检查网络连接后重试"，保留用户输入 |
| 密码校验接口失败 | 显示"网络异常，使用本地校验"，允许提交 |
| 密码修改失败 | 显示错误原因，保留表单数据 |

### 8.2 并发操作

| 场景 | UI处理 |
|------|--------|
| 多标签页同时刷新Token | 加锁，只允许一个请求 |
| 登录页面多次点击 | 按钮禁用，显示loading |
| 密码输入快速变化 | 防抖500ms后校验 |
