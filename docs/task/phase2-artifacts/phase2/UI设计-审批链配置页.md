## 三、交互规范

### 3.1 页面加载动画

- 筛选标签：从左到右依次淡入，delay 50ms
- 卡片列表：依次淡入上滑，delay 100ms
- 卡片出现动画：opacity 0→1，translateY 20px→0，300ms ease-out

### 3.2 筛选交互

- 点击标签：背景色过渡 200ms
- 筛选切换：列表淡入淡出 150ms

### 3.3 卡片交互

- 悬停：轻微上移 2px，阴影加深
- 点击编辑/删除：按钮区域高亮
- 删除按钮悬停：文字变红 #EF4444

### 3.4 弹窗动画

- 打开：背景淡入 200ms，弹窗缩放弹出 300ms cubic-bezier(0.34, 1.56, 0.64, 1)
- 关闭：弹窗缩放消失 200ms，背景淡出 200ms

### 3.5 表单验证

- 必填项未填：边框变红 #EF4444，底部显示错误提示
- 填写正确：边框变绿 #10B981

## 五、验收要点

1. 页面布局符合移动端规范，内容清晰可读
2. 筛选标签点击切换流畅，有选中状态视觉反馈
