**创建日期**：2026年4月13日
**状态**：待实施

## 一、背景与目标

### 1.1 背景
登录页当前存在以下问题：
- 特效堆叠过多（粒子、波浪、旋转环等），真机性能不佳
- 特效不够精美，部分动画卡顿
- 布局层次不够清晰，视觉重心模糊
- 需要为后续系统设计者提供可扩展的基础

### 1.2 目标
- 精简优化特效，提升真机流畅度
- 优化布局层次和字体层级
- 建立清晰的视觉重心，突出品牌感
- 保持企业专业风格

## 二、优化范围

**登录页 + 设计系统升级**

本次优化聚焦登录页，同时完善设计变量系统，为后续页面复用打基础。

## 三、特效优化方案

### 3.1 移除的特效

| 特效 | 原因 |
|------|------|
| 登录卡片入场动画 | 移除 scale + translateY 动画，页面加载即稳定呈现 |
| 标题逐字出现动画 | 移除 v-for + animationDelay，简化实现 |

### 3.2 精简优化的特效

| 特效 | 优化方案 |
|------|----------|
| 波浪动画 | 3层 → 1层，减少 GPU 合成层 |
| 粒子效果 | 30个 DOM 元素 → Canvas 实现，性能更好 |

**波浪动画优化代码**：
```css
.wave {
  position: absolute;
  bottom: 0;
  width: 200%;
  height: 60px;
  background: linear-gradient(90deg,
    transparent, rgba(0,82,217,0.25), transparent);
  animation: wave-move 12s linear infinite;
}
```

**粒子 Canvas 实现要点**：
- 创建 `<canvas>` 覆盖全屏
- 使用 `requestAnimationFrame` 驱动动画
- 粒子数据用 JS 数组管理（数量 15-30 可调）
- 绘制圆形粒子 + 上升动画

### 3.3 保留的特效

| 特效 | 说明 |
|------|------|
| Logo 旋转环 | 3层全部保留 |
| 按钮闪光 | ::before 伪元素扫光效果保留 |
| 输入框 hover/focus | 边框和背景色变化效果保留 |
| 鼠标光晕 | mousemove 事件保留（移动端无效但不影响） |
| 渐变背景 | 深色渐变背景保留 |

### 3.4 性能提升预估

| 指标 | 优化前 | 优化后 |
|------|--------|--------|
| DOM 节点 | ~45个 | ~16个 |
| GPU 合成层 | 多层 | 减少 |
| 低端机型表现 | 可能卡顿 | 流畅 |

## 四、布局层次优化

### 4.1 间距调整

| 元素 | 当前间距 | 优化间距 | 说明 |
|------|----------|----------|------|
| Logo → 标题 | 24px | 24px | 保持不变 |
| 标题 → 副标题 | 8px | 8px | 保持不变 |
| 副标题 → 卡片 | 36px | **48px** | 增大，拉开上下区域 |
| 卡片 → 版权 | 24px | 24px | 保持不变 |

### 4.2 布局结构

```
┌─────────────────────────────────────────────────────────────────┐
│  背景层（深色渐变 + Canvas 粒子 + 单层波浪）                      │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│                    Logo (110px)                                 │
│                  + 3层旋转环                                    │
│                         ↓ 24px                                  │
│                   标题"极智协同"                                 │
│                         ↓ 8px                                   │
│               副标题"高效协同 · 智慧办公"                        │
│                         ↓ 48px                                  │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │                     登录卡片                              │   │
│  │                  （宽度 580px）                           │   │
│  └─────────────────────────────────────────────────────────┘   │
│                         ↓ 24px                                  │
│                   底部版权 v1.0.0                               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 六、视觉重心设计

### 6.1 设计目标

Logo 和登录卡片同等重要，形成上下呼应。

### 6.2 视觉权重分配

| 元素 | 权重 | 实现方式 |
|------|------|----------|
| Logo 区域 | 高 | 大尺寸 Logo + 旋转环吸引注意力 |
| 登录卡片 | 高 | 毛玻璃效果 + 品牌色按钮形成焦点 |
| 背景 | 低 | 深色调，不喧宾夺主 |
| 粒子/波浪 | 低 | 半透明，增加氛围但不干扰 |

### 6.3 对齐原则

Logo 和卡片居中对齐，形成一条隐形的视觉中轴线。
