## 一、项目概述

### 1.1 项目背景

极智协同是一款面向企业职场场景的 H5 移动端办公应用，提供会议预定、差旅申请、请假打卡、文档查询、待办事项、公告通知等功能。

### 1.2 用户角色

| 角色 | 说明 |
|------|------|
| 普通员工 | 日常办公操作：会议预定、差旅申请、请假打卡、文档查询、待办管理 |
| 管理员 | 审批管理、会议室配置、文档管理、考勤管理、公告发布 |

## 二、技术架构

### 2.1 整体架构

```
┌─────────────────────────────────────────────────────────┐
│                      前端 (Vue 3)                        │
│                    aixt.coderzzp.top                     │
└─────────────────────────────────────────────────────────┘
                           │
                           │ HTTPS API
                           ▼
┌─────────────────────────────────────────────────────────┐
│                   后端 (FastAPI)                         │
│                  admin.coderzzp.top                      │
└─────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│                   数据库 (MySQL 8)                       │
└─────────────────────────────────────────────────────────┘
```

### 2.2 技术栈

#### 前端

| 类别 | 技术 |
|------|------|
| 框架 | Vue 3 + Composition API |
| 构建 | Vite 6 |
| 路由 | Vue Router 4 |
| 状态 | Pinia |
| UI | TDesign Mobile + Vant 4 |
| 样式 | TailwindCSS + Less |
| 请求 | Axios |

#### 后端

| 类别 | 技术 |
|------|------|
| 框架 | FastAPI |
| ORM | SQLAlchemy (异步) |
| 数据库 | MySQL 8 |
| 认证 | JWT |
| 部署 | Docker + Docker Compose |

## 四、部署方式

### 4.1 架构说明

项目采用 **Docker 容器化部署 + GitHub Actions 自动化 CI/CD**：

- 推送 `master` 分支自动触发部署
- 支持 SQL 数据库迁移自动执行
- 支持版本追踪
