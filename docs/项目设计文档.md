## 一、项目概述

### 1.1 项目背景

极智协同是一款面向企业职场场景的 H5 移动端办公应用，提供会议预定、差旅申请、请假打卡、文档查询、待办事项、公告通知、任务协作等功能。

### 1.2 用户角色

| 角色 | 说明 |
|------|------|
| 普通员工 | 日常办公操作：会议预定、差旅申请、请假打卡、文档查询、待办管理、任务协作 |
| 管理员 | 审批管理、会议室配置、文档管理、考勤管理、公告发布、数据统计 |

### 1.3 版本历史

| 版本 | 日期 | 说明 |
|------|------|------|
| v1.0 | 2026-04-12 | 一期基础功能上线 |
| v1.1 | 2026-04-21 | 交互优化与问题修复 |
| v2.0 | 2026-05-01 | 二期迭代完成：安全加固、审批增强、意见反馈、知识库、数据统计、任务协作 |

## 二、技术架构

```

│                      前端 (Vue 3)                        │
```

## 四、二期功能模块（安全加固 + 协作增强）

### 4.1 安全加固（第一阶段）

| 功能 | 说明 |
|------|------|
| 登录失败限制 | 连续5次失败锁定15分钟 |
| 双Token机制 | Access Token + Refresh Token |
| Token自动刷新 | 静默续期，无需频繁登录 |
| 密码强度校验 | 必须包含大小写字母、数字、特殊字符 |
| 密码过期提醒 | 密码90天过期，提前7天提醒 |
| 操作日志 | 记录登录和关键操作审计日志 |
| 请求频率限制 | 全局1000次/分钟，用户100次/分钟 |

### 4.2 审批增强 + 意见反馈（第二阶段）

| 功能 | 说明 |
|------|------|
| 自定义审批链 | 配置多级审批流程 |
| 批量审批 | 一次审批多个申请 |
| 审批进度查询 | 查看申请审批节点状态 |
| 催办提醒 | 申请人可催促审批 |
| 消息中心 | 审批通知统一管理 |
| 意见反馈 | 提交建议/Bug/问题反馈 |
| 反馈处理 | 管理员回复/关闭反馈 |

### 4.3 数据可视化 + 知识库 + 智能调度（第三阶段）

| 功能 | 说明 |
|------|------|
| 会议使用率统计 | 按日/周/月/季度统计会议室使用率 |
| 考勤分析报表 | 部门/个人考勤汇总分析 |
| 审批统计 | 各类型审批通过率、时效统计 |
| 综合数据大屏 | 核心指标一目了然 |
| 知识库 | 制度文档、技术指南、常见问题 |
| 知识分类 | 树形分类结构 |
| 知识搜索 | 关键词全文检索 |
| 智能推荐会议室 | 根据参会人数、时间推荐最佳会议室 |

### 4.4 任务协作（第四阶段）

| 功能 | 说明 |
|------|------|
| 任务看板 | 看板视图管理任务（待处理/进行中/已完成） |
| 任务详情 | 完整的任务描述、优先级、截止日期 |
| 多负责人 | 一个任务可分配给多人 |
| 子任务 | 分解大任务为子任务 |
| 任务评论 | 针对任务进行讨论 |
| 任务动态 | 记录所有变更历史 |
| 任务关注 | 关注任务获取更新通知 |
| 批量操作 | 批量更新状态/删除任务 |

## 七、部署方式

### 7.1 架构说明

项目采用 **Docker 容器化部署 + GitHub Actions 自动化 CI/CD**：

- 推送 `master` 分支自动触发部署
- 支持 SQL 数据库迁移自动执行
- 支持版本追踪

### 7.2 服务器配置

| 配置项 | 值 |
|--------|-----|
| 服务器 IP | 118.25.182.15 |
| 操作系统 | Ubuntu 22.04 LTS |
| 容器管理 | Docker + Portainer |

### 7.3 常用命令

```bash
# 前端开发
cd howtoai
npm run dev

# 前端构建
npm run build

# 后端开发
cd howtoai-backend
python -m uvicorn app.main:app --reload --port 8000

# 后端部署
docker-compose up -d
```
